Some information on the password protection of notes:
  • DO NOT USE a password you have used elsewhere, there is NO guarantee that the password cannot be discovered
  • Notes are not encrypted, just password protected to stop casual user access
  • You can not password protect a blank note and deleting all text from a note will also remove the password
  • Anyone who has access to the files on the server would be able to read the note if they wanted to
  • The password protection is only intended to give a very simple level of security
  • Want to store a secure note? Keep it somewhere else
  • NEVER store anything confidential in any note
